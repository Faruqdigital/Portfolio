As a Front End Developer I'm good at using:

Front-end developer plays a crucial role in creating websites and applications, and they are responsible for ensuring that the user interface and user experience are intuitive, engaging, and user-friendly.

  • HTML & CSS

    HTML provides the foundation for creating web pages and web applications.

    CSS (Cascading Style Sheets) is a language used for styling and designing the visual presentation of web pages.

  • BOOTSTRAP & TAILWIND

    Bootstrap provides a collection of CSS and JavaScript components that can be easily integrated into web projects, reducing the need for custom styling and coding.

    Tailwind is a utility-first CSS framework that allows developers to rapidly build custom user interfaces.

  • JAVASCRIPT

    JavaScript is a high-level programming language that is commonly used for creating interactive web pages and web applications. It is a versatile language that can be used on both the front-end and back-end of web development. JavaScript allows developers to create dynamic and responsive web content that can react to user actions and events.
